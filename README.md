# flask-online-shopify-api

---
This uses the following tech stack:
- Flask
- SQLAlchemy
- Flask Smorest
- Flask JWT
- Flask Migrate with Alembic
- Docker
- Gunicorn

---
It concentrates on three central model schema around:
- Stores
- Items
- Category
